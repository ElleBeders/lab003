# Задание 1

## Задача 1

### Текст задачи

На основе списка, содержащего римские обозначения цифр от I до IX, получить их десятичные представления.

### Алгоритм решения

Пользователем задаётся список из римских чисел от 1 до 9. Функция последовательно проверяет каждый элемент списка на соотетствие римским числам. Если элементу находится соответствие, то в последовательность вписывается арабская цифра. Если совпадения нет - в последовательность записывается 0. Когда обход списка и создание последовательности завершаются, они выводятся пользователю.

### Тестирование

Скриншоты результата работы программы


## Задача 2

### Текст задачи

Список содержит десятичные цифры. Составить число из чётных цифр.

### Алгоритм решения

Пользователем задаётся список из цифр, который преобразуется в числовую последовательность. Каждый элемен последовательности проверятся по правилу (x % 2 = 0). В реультате если число нечётное, оно пропускается, а если чётное, то "приписывается" к новому числу в качестве разряда. Когда все элементы проверены, программа выводит полученое из чётных цифр число.
